Step-by-step guides for getting API access to post content on each social media platform.

| Platform | Guide | Review Required | Timeline | Cost |

## Pre-Application Checklist (All Platforms)

Before submitting to any platform that requires review:

- [ ] Production-quality privacy policy covering data collection, use, retention, and deletion
- [ ] Public-facing website with real app description (scribario.com)
- [ ] Working app that a reviewer can log into and use
- [ ] Separate test credentials for reviewers (don't expose production data)
- [ ] Screen recording setup ready (OBS or Loom; 1080p minimum)
- [ ] Business entity documents ready (Meta and LinkedIn require these)
- [ ] Domain verified in Google Search Console (for YouTube)
- [ ] Use case description written: which users, what they do, why they need this permission

## Recommended Order

1. **Bluesky** — No review, instant access. Ship it first.
2. **X / Twitter** — Account approval + credits, fastest paid path.
3. **Meta** — Highest priority (Facebook + Instagram in one app). Start Business Verification immediately.
4. **TikTok** — Start sandbox testing while waiting for Meta.
5. **LinkedIn** — Requires legal org verification, longer timeline.
6. **YouTube** — Only if video-first posting is a priority.
