# Vibe Filmmaking with Veo 3.1 + Claude

**Source:** https://github.com/zechenzhangAGI/vibe-filmmaking-veo3-claude

## Workflow

Human Vision → Claude Code Collaboration → Veo Generation → Human Curation → Iterate → Final Production

## 5 Creative Pivots (V1-V5)

1. **V1: Photorealistic documentary** — "too literal" for manifesto storytelling
2. **V2: Apple-style simplified** — better emotional tone but still photorealistic
3. **V3: Balanced** — good narrative flow but lacked manifesto energy
4. **V4: BREAKTHROUGH** — tested 6 animation styles (watercolor, pencil sketch, vector, ink wash, 3D, paper cut-out). **Pencil sketch won** — "humanistic, hand-crafted, authentic"
5. **V5: Final refinement** — "camera movements that breathe," emotionally-landing character actions, match cuts

## Key Lessons

### Animation > Photorealism for Emotional Content
"Pencil sketch = humanity in AI-generated art." Less prescriptive prompting yielded SUPERIOR artistic interpretation.

### Prompt Structure (Claude Code recommended)
- Subject — what/who appears
- Action — what they're doing
- Style — visual aesthetic
- Camera — movement and framing
- Ambiance — lighting, mood, audio

### Less Prescriptive = Better
Detailed technical specs didn't help. Letting the model interpret creatively produced better results than over-specifying.

### Same Gesture, Opposite Meaning
The "eureka moment" — same hand gesture (sketching) conveying joy (Galileo) vs bureaucratic drudgery (modern researcher). Emotional contrast drives impact.

## Production Stats

- 5 iterations across 2 days
- 26 total clips (tests + production)
- ~850MB raw footage
- 100% prompt collaboration with Claude Code
- Final 1:17 video edited in CapCut
- 1080p, 8-second clips max
- 60-second rate limiting between API calls

## Critical Takeaway

"What once required an entire production team — writing, direction, cinematography, editing — can now emerge from one person, one laptop, and a well-constructed set of prompts."
