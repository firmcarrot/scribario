## Key Topics Covered

- **Camera Control:** Shot types, movements, what works vs fails, exact wording
- **Character Consistency:** Bible approach, verbatim descriptions, reference images
- **Audio/SFX:** Colon syntax, ambient specification, subtitle suppression
- **Multi-Clip Workflows:** Extend feature, first/last frame, sequential generation
- **Prompt Structure:** JSON vs prose, 100-150 word sweet spot, 7-element framework
- **Failure Modes:** Subject mutation, camera drift, cartoon audio, subtitle hallucination
- **Lighting:** Physical sources, temperature, key-to-fill ratios
- **Iteration:** Image-first validation, draft with fast mode, refine with full mode
